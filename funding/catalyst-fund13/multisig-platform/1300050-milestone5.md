## Milestone 5 Report

Hey there Milestone Reviewers, voters and really anyone who follows the progress of this proposal.
Here we are, reporting on the successful completion of the third proposal milestone. Took once again a bit longer than expected but we finally got everything done. So, here we are.

The acceptance criteria for this milestone has been stated as:
- Completed video recording to showcase all new features
- Licensed the tool and its features under an open source license on a public GitHub repository (DONE)
- Complete an easy step by step guide for users to use the multisig


And here the deliverables, some details and the respective links:

#### Video to showcase features
QS, the lead developer of the multisig made a video to summarise current main features which have been developed and publicly deployed, find the video at:
- https://youtu.be/-TuPP0jQ7UU 

#### Open source licencing
The Mesh Multisig is fully open source as promised, the respective github repository is hosted under a Apache 2.0 open source licence, find the licence at:
- https://github.com/MeshJS/multisig?tab=Apache-2.0-1-ov-file#readme 

#### Easy Guide
To help users get started, the frontend UI of the multisig has been reworked to make new user onboarding more streamlined and easy, an additional step-by-step Guide has been created and is currently accessible at:
- https://github.com/MeshJS/governance/blob/main/funding/catalyst-fund13/multisig-platform/m5-stepbystep-guide.md 


And thats it, slowly wrapping up and closing thios proposal with the next milestone where we see each other again soon.
